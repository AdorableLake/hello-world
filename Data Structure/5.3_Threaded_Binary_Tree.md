# 5.3 Threaded Binary Tree | 线索二叉树
## 5.3.1 Inorder Traversal Queue in Binary Tree | 二叉树的中序遍历序列

```C
// Binary Tree in Link structure
typedef struct biTNode {
    ElemType data;
    struct biTNode *lChild, *rChild;
} biTNode, *biTree;

// Threaded Binary Tree Node
typedef struct threadNode {
    ElemType data;
    struct threadNode *lChild, *rChild;

    int lTag, rTag;     
    // Different from the normal binary tree

} threadNode, *threadTree;

// tag==0 => ChildNode;
// tag==1 => ThreadPrior;

// Inorder Traversal
void inOrder(biTree T) {
    if(T != NULL) {
        inOrder(T->lChild);
        visit(T);
        inOrder(T->rChild);
    }
}
```

## 5.3.2 Thread Binary Tree | 线索化二叉树

1. Inorder Thread | 中序线索化

```C
// Threaded Binary Tree Node
typedef struct threadNode {
    ElemType data;
    struct threadNode *lChild, *rChild;
    int lTag, rTag;     
} threadNode, *threadTree;

// Global Variable
threadNode *pre = NULL;

// Inthread Tree
void inThread(threadTree T)
{
    if(T != NULL) {
        inThread(T->lChild);
        visit(T);
        inThread(T->rChild);
    }
}

void visit(threadNode *q) 
{
    if(q->lChild == NULL) {
        q->lChild = pre;
        q->lTag = 1;
    }

    if(pre != NULL && pre->rChild == NULL) {
        pre->rChild = q;
        pre->rTag = 1;
    }

    pre = q;
}
```
