# 3.1 栈/Stack
## 3.1 1 栈的相关概念
### 1. 术语定义
1. 栈
2. 栈顶
3. 栈底
4. 空栈

### 2. 基本操作
创建/初始化👇
```
InitStack(*S)
```
销毁👇
```
DestroyStack(*S)
```
增加/进栈👇
```
Push(*S,x)
```
删除/出栈👇
```
Pop(*S,*x)
```
访问栈顶元素👇
```
GetTop(S,*x)
```
判空👇
```
StackEmpty(S)
```
