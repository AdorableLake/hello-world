答：

```C

{

  printf("Enter the number:\n");

  res=sign(x);
  printf("The result is %d.\n",res);
  return 0;

int sign(x)
{
  int production;
  if(x>0)
  {
    production=1;
  }
  else if(x==0)
  {
    production=0;
  }
  else if(x<0)
  {
    production=-1;
  }
  return (production);
}
```

3. 使用函数计算两点间的距离：给定平面任意两点坐标(x1,y1)和(x2,y2)，求这两点之间的距离（保留两位小数）。要求定义和调用函数dist(x1,y1,x2,y2)计算两点间的距离。

```C
  double x1,y1,x2,y2;
```
